# CODED Build Plan

## Phase 1 — Engine Shell

- Vite + TypeScript static app.
- Canvas 2D renderer.
- Title screen, play state, game-over restart.
- Keyboard and touch movement.
- Auto-fire projectile loop.
- Basic enemy wave spawning.
- Projectile/enemy collision.
- Placeholder clarity pulse special ability.
- HUD with score, wave, health, and special meter.
- Vercel-ready static build.

## Phase 2 — Asset Integration

- Add optimized ship, enemy, projectile, UI, and VFX assets.
- Expand manifest IDs.
- Add sprite-sheet support.
- Add asset diagnostics panel.

## Phase 3 — Boss Tech

- Add boss intro, phase system, special weapon unlocks, and upgrade-tree UI.
