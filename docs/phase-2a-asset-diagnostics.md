# Phase 2A Asset Diagnostics

## Added

- Build workflow scaffold.
- Improved start flow using one-time tap consumption.
- Mobile HUD zones for pause, pulse, and diagnostics.
- Asset diagnostics overlay.
- Manifest schema support for image and spritesheet entries.
- Missing asset console reporting.
- Safe optimized SVG placeholder assets.

## Controls

- Drag anywhere outside HUD controls to move.
- Tap PULSE to trigger the special meter when full.
- Tap PAUSE to pause.
- Tap D or press keyboard D to toggle diagnostics.
